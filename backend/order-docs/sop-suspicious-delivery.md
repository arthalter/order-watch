# SOP：可疑收货信息/地址异常处理

## 适用场景
- 收货地址/收件信息存在异常特征，疑似转运、虚假地址或难以联系，导致履约风险。

## 判定标准（常用）
- 地址过于模糊（缺少门牌、楼栋、电话等）
- 地址/收件人多次修改，且修改频繁
- 地址首次出现且与用户常用地区差异大
- 收件人/电话异常（无效号段、明显随机）

## 处理步骤（最小流程）
1. 暂缓发货（进入人工核验）。
2. 校验地址完整性（是否可配送、是否可联系）。
3. 联系买家补全信息（补全后再发货；沟通内容留痕）。
4. 如无法联系或明显异常：取消/拦截订单并记录原因。

## 注意事项
- 处理目标是降低履约损失，优先“补全信息”而不是直接取消。
- 所有联系与结论要留痕，便于复盘与申诉处理。
